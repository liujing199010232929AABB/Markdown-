# Lesson03 标题
-----------------------------------------------------------------------------------

## 1.标题使用的标记是："#"号
-----------------------------------------------------------------------------------

## 2.实战演示：
# 一级标题 #
## 二级标题 ##
### 三级标题 #
#### 四级标题 #
##### 五级标题 #
###### 六级标题 #
说明：可以选择相等数量的#进行闭合
-----------------------------------------------------------------------------------

## 3.对应的html标签:&lt;h1&gt; ~&lt;h6&gt;
-----------------------------------------------------------------------------------
html实体符号：防止浏览器进行解析，会将它所代表的字符原样显示，例如：&lt;h1&gt;最终显示为：<h1>


### 4.一、二级标题的简写
- 一级标题，在文本下面添加一个或以上的等号:===
- 二级标题，在文档下面添加一个或以上的减号:---

php中文网
======================================================

www.php.cn
------------------------------------------------------
